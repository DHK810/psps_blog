🎵 근음 & 어울리는 음

근음 & 어울리는 음 찾기

코드의 근음(Root Note)을 기준으로 어울리는 음을 알면, 코드 구성 원리와 솔로 연주를 이해하는 토대가 됩니다.

근음(Root Note)이란?

코드 이름의 기준이 되는 음입니다. C코드의 근음은 C, Am코드의 근음은 A입니다. 근음 위에 다른 음들을 쌓아 코드를 만들죠.

메이저 코드
1 + 3 + 5
근음 + 장3도(4반음) + 완전5도(7반음)
밝고 안정적인 소리
vs
마이너 코드
1 + ♭3 + 5
근음 + 단3도(3반음) + 완전5도(7반음)
어둡고 감성적인 소리
핵심: 메이저↔마이너의 차이는 단 반음 하나. 3도 음이 반음 내려가면(장3도 → 단3도) 마이너가 됩니다.

근음 선택 → 어울리는 음 확인

① 근음 선택
② 스케일 / 코드 유형 선택

6번줄(E) · 5번줄(A) 음 이름 — 바레 코드 근음 찾기

바레 코드는 근음 줄의 프렛 위치가 곧 코드 이름이 됩니다. 이 두 줄의 음 이름만 외우면 프렛보드 전체를 활용할 수 있습니다.

6번줄 (E string) — E형 바레 코드 근음
5번줄 (A string) — A형 바레 코드 근음
활용 예시:
E형 바레를 3프렛에 → G코드 / 5프렛에 → A코드 / 8프렛에 → C코드
A형 바레를 2프렛에 → B코드 / 5프렛에 → D코드 / 7프렛에 → E코드

12음 반음 구조 이해하기

기타의 한 프렛 = 반음 1개. 이 구조를 알면 모든 음 위치를 스스로 계산할 수 있습니다.

    근음 찾기: 6번줄 또는 5번줄에서 코드 이름에 해당하는 프렛 위치를 찾는다
  • 2
    코드 음 계산: 근음에서 위 도표대로 4반음(메이저) 또는 3반음(마이너) 올리면 3도 음
  • 3
